# Методы массивов

### 1. `map`

**Описание:** Создает новый массив с результатом вызова указанной функции для каждого элемента массива.

**Особенности:**

- Не изменяет исходный массив
- Возвращает новый массив той же длины
- Пропускает пустые элементы (undefined)
- Принимает callback-функцию с аргументами (элемент, индекс, исходный массив)

**Пример:**

```typescript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map((num) => num * 2);
// [2, 4, 6, 8]
```

### 2. `filter`

**Описание:** Создает новый массив со всеми элементами, прошедшими проверку в callback-функции.

**Особенности:**

- Не изменяет исходный массив
- Возвращает новый массив, который может быть короче исходного
- Пропускает пустые элементы (undefined)
- Принимает callback-функцию, которая должна возвращать булево значение

**Пример:**

```typescript
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter((num) => num % 2 === 0);
// [2, 4]
```

### 3. `reduce`

**Описание:** Применяет функцию к аккумулятору и каждому элементу массива (слева направо), сводя его к одному значению.

**Особенности:**

- Не изменяет исходный массив
- Возвращает одно значение (любого типа)
- Принимает callback-функцию с аргументами (аккумулятор, текущий элемент, индекс, исходный массив)
- Принимает начальное значение аккумулятора (если не указано, используется первый элемент)
- Является наиболее гибким методом, можно реализовать функциональность map и filter

**Пример:**

```typescript
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, current) => acc + current, 0);
// 10
```

### 4. `forEach`

**Описание:** Выполняет указанную функцию один раз для каждого элемента массива.

**Особенности:**

- Не создает новый массив
- Всегда возвращает undefined
- Нельзя прервать выполнение (в отличие от for/while циклов)
- Изменения в исходном массиве возможны, но не рекомендуются
- Нельзя использовать с async/await

**Пример:**

```typescript
const numbers = [1, 2, 3, 4];
numbers.forEach((num) => console.log(num));
// 1, 2, 3, 4 (в консоли)
```

### 5. `find`

**Описание:** Возвращает первый элемент массива, который удовлетворяет условию проверяющей функции.

**Особенности:**

- Не изменяет исходный массив
- Возвращает элемент массива (не индекс)
- Возвращает undefined, если элемент не найден
- Останавливает выполнение после нахождения первого подходящего элемента
- Не пропускает пустые элементы

**Пример:**

```typescript
const numbers = [1, 5, 10, 15];
const found = numbers.find((num) => num > 7);
// 10
```

### 6. `some`

**Описание:** Проверяет, удовлетворяет ли хотя бы один элемент массива условию в callback-функции.

**Особенности:**

- Не изменяет исходный массив
- Возвращает булево значение (true/false)
- Прекращает выполнение при первом элементе, удовлетворяющем условию
- Для пустого массива всегда возвращает false

**Пример:**

```typescript
const numbers = [1, 2, 3, 4, 5];
const hasEven = numbers.some((num) => num % 2 === 0);
// true
```

### 7. `every`

**Описание:** Проверяет, удовлетворяют ли все элементы массива условию в callback-функции.

**Особенности:**

- Не изменяет исходный массив
- Возвращает булево значение (true/false)
- Прекращает выполнение при первом элементе, не удовлетворяющем условию
- Для пустого массива всегда возвращает true

**Пример:**

```typescript
const numbers = [2, 4, 6, 8];
const allEven = numbers.every((num) => num % 2 === 0);
// true
```

### 8. `findIndex`

**Описание:** Возвращает индекс первого элемента массива, удовлетворяющего условию.

**Особенности:**

- Не изменяет исходный массив
- Возвращает индекс или -1, если элемент не найден
- Останавливается после нахождения первого совпадения

**Пример:**

```typescript
const fruits = ["apple", "banana", "cherry"];
const index = fruits.findIndex((fruit) => fruit === "banana");
// 1
```

### 9. `includes`

**Описание:** Определяет, содержит ли массив определенный элемент.

**Особенности:**

- Не изменяет исходный массив
- Возвращает булево значение (true/false)
- Может принимать второй параметр - индекс, с которого начинать поиск
- Использует строгое сравнение (===), но работает корректно с NaN

**Пример:**

```typescript
const numbers = [1, 2, 3, 4, 5];
const hasThree = numbers.includes(3);
// true
```

### 10. `slice`

**Описание:** Возвращает новый массив, содержащий копию части исходного массива.

**Особенности:**

- Не изменяет исходный массив
- Принимает параметры start и end (необязательный)
- Поддерживает отрицательные индексы (отсчет с конца массива)

**Пример:**

```typescript
const numbers = [1, 2, 3, 4, 5];
const sliced = numbers.slice(1, 4);
// [2, 3, 4]
```

### 11. `splice`

**Описание:** Изменяет содержимое массива, удаляя или заменяя существующие элементы и/или добавляя новые.

**Особенности:**

- Изменяет исходный массив
- Возвращает массив удаленных элементов
- Может использоваться для удаления, добавления и замены элементов

**Пример:**

```typescript
const months = ["Jan", "March", "April", "June"];
months.splice(1, 0, "Feb");
// месяцы теперь ['Jan', 'Feb', 'March', 'April', 'June']
```

### 12. `sort`

**Описание:** Сортирует элементы массива на месте и возвращает отсортированный массив.

**Особенности:**

- Изменяет исходный массив
- По умолчанию сортирует элементы как строки
- Для правильной сортировки чисел необходимо передать функцию сравнения
- Сложность O(n log n)

**Пример:**

```typescript
const numbers = [4, 2, 5, 1, 3];
numbers.sort((a, b) => a - b);
// [1, 2, 3, 4, 5]
```

### 13. `flat`

**Описание:** Создает новый массив с рекурсивно объединенными подмассивами до указанной глубины.

**Особенности:**

- Не изменяет исходный массив
- Принимает параметр глубины (по умолчанию 1)
- Удаляет пустые слоты из результата

**Пример:**

```typescript
const nestedArray = [1, 2, [3, 4, [5, 6]]];
const flattened = nestedArray.flat(2);
// [1, 2, 3, 4, 5, 6]
```

### 14. `flatMap`

**Описание:** Комбинация map() и flat() глубиной 1 - сначала применяет функцию к каждому элементу, а затем сглаживает результат.

**Особенности:**

- Не изменяет исходный массив
- Более эффективен, чем раздельное использование map().flat()
- Часто используется для обработки данных и одновременного выравнивания результатов

**Пример:**

```typescript
const sentences = ["Hello world", "JavaScript is awesome"];
const words = sentences.flatMap((sentence) => sentence.split(" "));
// ["Hello", "world", "JavaScript", "is", "awesome"]
```

### 15. `from`

**Описание:** Создает новый экземпляр массива из массивоподобного или итерируемого объекта.

**Особенности:**

- Статический метод (Array.from)
- Позволяет преобразовать объекты, такие как NodeList, arguments и итерируемые объекты в массивы
- Принимает опциональную функцию map как второй аргумент

**Пример:**

```typescript
// Преобразование строки в массив символов
const chars = Array.from("hello");
// ['h', 'e', 'l', 'l', 'o']

// Преобразование с функцией map
const numbers = Array.from([1, 2, 3], (x) => x * 2);
// [2, 4, 6]
```
